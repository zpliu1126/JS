讲一讲如何使用github将我的代码使用js展示吧！

  • 首先你得有你的github的 主页
  • 接着再创建一些仓库的时候，把html页面放在gh-page分支下，才能够进行访问
  • 把主要的CSS JS 文件放在github主页的同样名称的文件夹下；当自己在本地运行成功时直接上传就行了
  • 我的是centos+appach建的站，所以在/var/www/html/下建立同样的CSS JS 文件夹

关于如何进行服务器端上传github文件的流程

  • 初始化仓库git init
  • 连接远程仓库 git remote add origin SSH地址
  • 要使用gh-pages分支进行展示，创建分支 git branch gh-pages
  • 选择分支 git checkout gh-pages
  • 拉下github端的数据放在gh-pages分支 git pull origin gh-pages
  • 添加上传文件 git add ./*
  • 说明 git commit -m "说明事件发生"
  • 开始上传 git push origin gh-pages
